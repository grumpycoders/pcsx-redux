# Emulator compatibility

## Emulators shipping with OpenBIOS
* PCSX PCSX-Redux
* PCSX ReARMed

## Other emulators compatible with OpenBIOS

RetroArch cores:
* Beetle PSX HW
* Beetle PSX
* PCSX ReARMed
* SwanStation

# Incompatible games
OpenBIOS is compatible with most games. Please report any titles that fail to run so they can be added to this section.
